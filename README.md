# Digital Forensics: File Integrity Checker 🛡️

A professional Python-based tool designed to verify the integrity of digital evidence using the *SHA-256* hashing algorithm. This tool is essential for Digital Forensics investigators to ensure that files have not been tampered with.

## 🚀 Features
- *Data Integrity:* Generates a unique digital fingerprint (Hash) for any file.
- *Avalanche Effect:* Demonstrates how a tiny change in data results in a completely different hash.
- *Forensic Standard:* Uses SHA-256, a globally recognized standard in cybersecurity.
